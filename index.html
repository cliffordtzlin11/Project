Our Partners

Success in our task is only possible if we work together towards a common goal.

PopSockets® Recycling Program

PopSockets has partnered with TerraCycle to create the PopSockets Recycling Program, a free program that accepts old phone cases from any brand as well as PopSockets products and packaging for recycling. Learn how to send in old cases and PopSockets.

Close the Loop Recycling Program

Close the Loop partners with Case-Mate, Incipio and Survivor to offer phone case recycling (any brand). The Case-Mate program also accepts phone charging accessories.
